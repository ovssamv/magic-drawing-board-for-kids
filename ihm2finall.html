How to Use the Drawing Board

  1. Choose a tool from the toolbar
  2. Select a color using the color picker
  3. Adjust line width using the slider
  4. Click and drag on the canvas to draw
  5. Save your work as draft to continue later
  6. Use Ctrl+Z to undo and Ctrl+Shift+Z to redo
